## What is HyzeCloud?

HyzeCloud is a modern platform for deploying and managing applications with ease. Whether you're running Node.js, Python, or Bun applications, HyzeCloud provides a seamless experience for containerized deployments.

## Key Features

- **Multi-Runtime Support**: Native support for Node.js, Python, and Bun.
- **Instant Deployments**: Deploy from ZIP files or Git repositories.
- **Real-time Logs**: Monitor your application logs in real-time.
- **Environment Management**: Securely manage environment variables.
- **Backup & Restore**: built-in backup solutions for your data.
